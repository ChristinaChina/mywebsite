## Basic Data Manipulation

### View Data
```r
head(data)
tail(data)
str(data)
summary(data)
```

### Selecting and Filtering
```r
data$age
data[, "age"]
data[1:5, ]
subset(data, age > 25)
```

### Adding and Modifying Columns
```r
data$age2 <- data$age * 2
```

### Sorting
```r
data[order(data$age), ]
data[order(-data$age), ]
```
