## Data Import and Export

### Read CSV File
```r
data <- read.csv("data.csv", header=TRUE)
```

### Write CSV File
```r
write.csv(data, "output.csv", row.names=FALSE)
```

### Read Excel File (requires readxl package)
```r
install.packages("readxl")
library(readxl)
data <- read_excel("data.xlsx", sheet=1)
```
