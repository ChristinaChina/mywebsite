## Data Visualization

### Base R Plots
```r
# Scatter plot
plot(data$age, data$salary)

# Histogram
hist(data$age)

# Boxplot
boxplot(data$age ~ data$group)
```

### ggplot2 Visualization
```r
install.packages("ggplot2")
library(ggplot2)

ggplot(data, aes(x=group, y=age, fill=group)) +
  geom_boxplot() +
  theme_minimal()
```
