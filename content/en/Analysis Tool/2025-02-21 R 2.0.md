## Basic Data Types and Variables

### Data Types

- Numeric: x <- 10
- Character: name <- "KC"
- Logical: flag <- TRUE
- Vector: v <- c(1, 2, 3)
- Factor: group <- factor(c("A", "B"))
- List: mylist <- list(num=1, name="KC")
- Data Frame: df <- data.frame(name=c("Alice","Bob"), age=c(25,30))
